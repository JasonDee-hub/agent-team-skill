# Agent Team 示例交付样本

这些样本用于说明什么叫合格的 Agent Team 输出：主控先给可判定派遣包，专家用固定字段回报，最终收口逐项对齐验收标准。

样本不是强制逐字模板；真实任务可以更短，但不能省掉权限、验收、证据和阻塞说明。
